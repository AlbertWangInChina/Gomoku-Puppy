# AI五子棋游戏Gomoku-Puppy(Java)

## 这里有什么

本篇文章的内容有:一个用Java做的简易五子棋游戏和五子棋AI.(附源代码,GitHub仓库URL)

## 为什么要做

我是一个Java自学者,刚刚看完了《Java核心技术卷一》,想先试试手.

恰巧个人比较喜欢五子棋,所以决定开发一个单机五子棋小游戏,附带一个简单的AI.

命名为Gomoku-Puppy,Gomoku即五子棋,Puppy是AI的名字(因为我喜欢小狗).
